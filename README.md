## Install 🔧 
1. Clone this repo.<br>
    ```git clone https://github.com/caminodelaserpiente/guia_librerias_paquetes_modulos.git```

2. Change the working directory.<br>
    ```cd guia_librerias_paquetes_modulos```

3. Use<br>
    ```python3 main.py```
